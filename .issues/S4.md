## What to build

实现 Web 管理后台的 B 端用户认证系统，支持两种登录方式：

- **账号密码登录**：B 端用户注册时填写手机号、密码（加密存储），登录时校验。
- **微信扫码登录**：接入微信开放平台网站应用扫码登录，场馆管理员使用微信扫码即可登录管理后台。

需要实现：
- Web 端登录页面（账号密码表单 + 微信扫码入口）
- 云函数处理注册（密码哈希存储）和登录（返回 token/session）
- 微信扫码登录 OAuth 流程（获取 code → 换取 access_token → 获取用户信息 → 关联或创建账户）
- 会话管理（登录态保持、登出）

## Acceptance criteria

- [ ] B 端用户可通过手机号 + 密码注册并登录
- [ ] B 端用户可通过微信扫码登录管理后台
- [ ] 密码存储使用哈希（bcrypt），不存明文
- [ ] 登录态有效期内可访问管理后台，过期后需重新登录
- [ ] 登出后清除登录态，无法访问受保护页面
