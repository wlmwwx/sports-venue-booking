## What to build

实现 C 端用户通过微信小程序一键授权登录的完整链路。

- 小程序端调用 `wx.login` 获取临时 code
- 云函数 `login` 接收 code，调用微信 `code2Session` API 换取 openid
- 将 openid 写入 `users` 集合（首次登录创建记录，后续返回已有用户信息）
- 小程序端获取登录态并存储，后续请求携带鉴权信息

不涉及手机号绑定、不涉及用户资料编辑页面 —— 只要能完成静默登录即可。

## Acceptance criteria

- [ ] 用户首次打开小程序，自动完成登录，无需手动操作
- [ ] 云函数成功获取 openid 并写入 users 表
- [ ] 同一 openid 多次登录返回同一用户记录，不重复创建
- [ ] 小程序端可获取当前用户信息
