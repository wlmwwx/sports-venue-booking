## What to build

C 端用户在选定场馆后，查看场地号和可用时段的页面。

**日期选择**：
- 日历组件，可选今天起未来 N 天（N 由场馆配置的提前预定天数上限决定）
- 不可预定日期灰显

**场地号选择**：
- 横向滑动或 Tab 切换不同场地号（1号场、2号场...）
- 每个场地号显示类型标签（室内/室外、木地板等）

**时段网格**：
- 以时间轴或网格展示当天该场地号的所有时段
- 每个时段格显示：时间、价格、状态（可预定/已预定/不可用）
- 已预定和不可用的时段灰显不可点击
- 可预定时段高亮，点击后进入预定确认

## Acceptance criteria

- [ ] 日历组件可正常翻页选日期，超出范围灰显
- [ ] 场地号切换流畅，数据正确
- [ ] 时段网格正确展示可预定/已预定/不可用状态
- [ ] 支持半小时和自定义时长的时段展示
- [ ] 不同价格时段正确显示对应金额
