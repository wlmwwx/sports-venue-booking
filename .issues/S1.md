## What to build

搭建整个项目的三端基础设施，打通本地开发到云端部署的完整链路。

- **微信小程序**：使用微信开发者工具创建原生小程序项目，配置 app.json，初始化目录结构。
- **微信云开发**：开通云开发环境，初始化云函数目录和云数据库。
- **Web 管理后台**：使用 Next.js 初始化项目，集成微信云开发 Web SDK。

三端能通过云开发环境连通即可，不需要构建任何业务功能。

## Acceptance criteria

- [ ] 微信小程序项目创建成功，能在开发者工具中运行
- [ ] 云开发环境已开通，云函数能本地调试
- [ ] Next.js 管理后台项目创建成功，npm run dev 可访问
- [ ] 管理后台能通过 Web SDK 连接到云开发环境

## Blocked by

None - 可立即开始
