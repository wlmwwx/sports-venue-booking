## What to build

C 端小程序首页 —— 用户浏览和发现运动场馆。

**场馆列表**：
- 以卡片列表形式展示已上架场馆
- 每张卡片展示：封面图、名称、运动类型标签、距离、最低价格

**筛选**：
- 顶部 Tab 按运动类型筛选（全部/羽毛球/篮球/网球/游泳等）
- 支持按距离排序（由近到远）

**地图视图**：
- 切换为地图模式，在地图上展示场馆位置
- 点击标记跳转场馆详情

**场馆详情页**：
- 展示完整信息：轮播图、地址（可唤起地图导航）、营业时间、联系电话（可拨打）
- 展示该场馆下所有场地号列表及属性

## Acceptance criteria

- [ ] 首页展示已上架场馆列表
- [ ] 按运动类型筛选有效
- [ ] 按距离排序正确（获取用户位置授权后）
- [ ] 地图模式下可看到场馆标记
- [ ] 场馆详情页展示完整信息
- [ ] 点击电话可拨出，点击地址可导航
