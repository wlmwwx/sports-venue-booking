# S5a：场馆入驻审核流程设计

## 入驻表单字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 场馆名称 | 文本 | 必填 | 30 字以内 |
| 运动类型 | 多选标签 | 必填 | 羽毛球/篮球/网球/游泳/足球/乒乓球/健身/其他 |
| 详细地址 | 文本 + 地图选点 | 必填 | 输入地址后自动定位，可手动调整经纬度 |
| 营业时间 | 时间范围 | 必填 | 如 08:00-22:00 |
| 联系电话 | 手机号 | 必填 | 用户可见的联系方式 |
| 场馆照片 | 图片上传 | 至少 1 张 | 最多 6 张 |
| 场馆简介 | 多行文本 | 选填 | 200 字以内 |

## 状态流转

场地方提交 -> 待审核 -> 平台审核 -> 通过(已上架) / 驳回(已驳回)
已上架 -> 平台下架 -> 已下架
已驳回 -> 场地方修改重提 -> 待审核

## 审核页面要素

- 审核列表：按提交时间倒序，显示场馆名称、运动类型、状态
- 审核详情：左侧场馆信息（含照片），右侧操作区
- 通过：一键确认，场馆立即上架
- 驳回：必填驳回原因，场地方可查看

## 驳回/重提规则

- 驳回次数不限
- 修改后重新提交，状态回到待审核
- 驳回原因对场地方可见
