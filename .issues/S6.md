## What to build

B 端在 Web 管理后台管理自己场馆的场地号和时段配置。

**场地号管理**：
- 在已上架场馆下添加场地号（如"1号场"），设置属性：室内/室外、地板材质、容纳人数
- 场地号列表、编辑、删除

**时段配置**：
- 为每个场地号设置每日可预定时间段（如 08:00-22:00）
- 支持半小时和自定义时长
- 支持不同日期的差异化价格（工作日/周末/黄金时段）
- 批量设置某天所有时段为不可预定（闭馆日/包场日）

**云函数**：
- 按配置规则批量生成/更新 time_slots 记录

## Acceptance criteria

- [ ] 可为场馆添加多个场地号并设置属性
- [ ] 可为每个场地号配置每日营业时段和价格
- [ ] 支持半小时及自定义时长
- [ ] 支持工作日/周末/节假日差异化定价
- [ ] 可批量设置某日所有时段为不可预定
- [ ] 时段数据可被 C 端查询接口正常读取
